Quadtree Painter

This is a quick one-day project for me to experiment with quadtrees. Click anywhere on the canvas to add a point to the quadtree. Upon adding a fifth point, the main tile will overflow and subdivide into 4 quarter-tiles. This process can be repeated to add finer detail.

The tiles are colored by the average color of all points in the tile. If edited carefully, one can make interesting patterns. To create a random pattern, simply press Enter a bunch of times.

Controls:

  • Click - Add a point to the canvas
  • Ctrl + Click - Remove all points for the quadtree leaf node under the cursor
  • Enter - Add a randomly colored and position to the canvas
  • Q, W, E - Select a point color by toggling red/green/blue channels
  • 1, 2, 3 - Toggle display of leaf node background color, boundaries, and points
  • P - Save a screenshot
  • Esc - Clear canvas
